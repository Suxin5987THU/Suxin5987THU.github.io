设置github pages

登录自己的github账号，建立对应的仓库。
这里必须要注意名字必须取为(username).github.io
注意是必须，否则会无法显示出博客网页。
然后在该仓库下创建hexo分支，并且将该分支设置为默认分支

设置hexo和相关配置

由于是第一次配置，所以要初始化：

在初始化之后就要配置相关的配置文件：_config.yml
这里主要要更改deploy选项

deploy:
 type: git
 repository: git@github.com:Suxin5987THU/Suxin5987THU.github.io.git
 branch: master
这里注意冒号之后的空格，否则会出错。
